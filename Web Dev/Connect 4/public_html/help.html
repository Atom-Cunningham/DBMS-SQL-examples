Connect4Verse Help Page

  • Press the back buttonin your browser from this page at any time to return to your session

  • To login, simply type your username and password and click login

  • If you do not have a username and password, type create account and type your intended username and password into the boxes, then click create account.

  • To play, once in the game lobby, click create game. If a game exists, you can join that game by clicking join next to the player name.
  • In the extremely unlikely event that you do not know how to play connect 4, click here.

  • IMPORTANT: In this version of connect 4, the user who created the game always goes first. The first player has the red pieces.

  • While playing the game, you can send a message to your opponent by entering a message in the box in the top right, and clicking the "send message" button

  • When one player wins, a message will be displayed for both players letting you them know if they won or lost. A button will also become available to redirect the player to the lobby.
